Recommendation Minimum-effort, maximum-clarity combination
Names → Option B First name only. Trivial split on " " in the template. Most people go by first name.
Scroll → Option C or D C: "+N" badge on arrow — one CSS addition, lowest effort.
D: ‹ › buttons — explicit, always visible, best for touch/keyboard. Costs 56px strip width.
Size → Option A Adaptive 28×38px when >10. Keeps all 20 visible without scroll. Combine with first-name-only.
